Web Basics Introduction

Web development:

Refers to develop for web ( internet based ) run on http protocol

Web development includes
Web Application
Web Service

Web Application

It is an application hosted in web server, where clients communicate via network

Web Server

It is a centralized server where applications and services are hosted

Advantages:
a) Application or service available round the globe
b) Application or service available round the clock
c) Client and server architecture

Types of web applications:

Server Side Web Application:

It is an application hosted in server, client request using HttpRequest object, server process the request and respond to the client using HttpResponse object These application available in remote Server side statement management techniques will maintain state Technologies like Java, .NET, Python, Node.js,etc., used to develop server side web applications

Client side Web Application

It is an application hosted in server client request using HttpRequest object, serer repond UI application to the client The entire application deployed in the browser The UI application even accessible at offline Technologies like Angular, React, Vue, backbone, etc., used to develop client side applications Web Service: It is an interoperable service avialable in web server, the other application developed using any technology can consume Web Service

Different types of web services:

a) SOAP Services: These are services consumed by any technology using SOAP protocol The service will be exposed using WSDL file Client consume using SOAPClient object Data shared via XML format

b) Web API : These are RESTful Services, can be consumed by any application without any restrictions Developed via HTTP terms

Web Client: It is is a browser, used to access web applications
